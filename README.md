PRODIGY_SE_02 - Number Guessing Game

Task Description -
Build a Java program that generates a random number between 1 and 100 and challenges the user to guess it.
The program:
Accepts user guesses

Provides feedback: "Too high", "Too low", or "Correct" & Continues until the correct number is guessed. Then displays the total number of attempts made.

Technologies Used

Language: Java

IDE: IntelliJ IDEA

Input/Output: Console

Random Number Generator: java.util.Random

Features
1. Random number generation between 1 to 100
2. Validates user guesses with instant feedback
3. Tracks and displays number of attempts
4. Clean and interactive console interface
